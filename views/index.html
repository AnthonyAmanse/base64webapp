Quickly Decode Base64 in your Browser!

Welcome!

Use this web app to quickly decode base64!

To decode, add the path below to this website's url
    /decode/(the base 64 encoded text you want to decode)
     ex. /decode/dGhpc2lzdGhldGV4dGRlY29kZWQ=

To encode, add the path below to this website's url
    /encode/(the text you want to encode in base64)
     ex. /encode/thisisthetexttobeencoded
